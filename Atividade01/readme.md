# Instruções

## 1 - Dependencias

- `tabulate`
- `numpy`
- `matplotlib`

### 1.1 - Sugestão de instalação

- Utilizando o instalador `pip`
- Execute os comandos:
  - `pip install tabulate`
  - `pip install numpy`
  - `pip install matplotlib`

## 2 - Executando os arquivos

```
python <nome_do_arquivo.py>
```
